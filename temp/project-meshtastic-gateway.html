Weatherproof Meshtastic Home Gateway

To establish a permanent local footprint in the Ozarks mesh, I am building a fixed MQTT home gateway node mounted high up on an outdoor mast to maximize coverage.

Key Build Details:
  • Hardware: Heltec V4 LoRa board running on 915 MHz.
  • Enclosure: Weatherproof IP67 outdoor junction box with customized internal 3D-printed brackets to secure the board and battery.
  • Power: Powered via a Power-over-Ethernet (PoE) split line or solar/LiFePO4 battery array for uninterrupted uptime.
